Task 2: 학과 찾기 데이터 연동 (29,498개 대학-학과 매핑)

목표: 업로드된 학과 찾기.xlsx 데이터를 DB에 반영하여 "이 전공을 어느 대학에서 제공하는지" 연결
데이터 파일: 프로젝트 폴더에 이미 업로드된 학과 찾기.xlsx

29,498개 행 (대학-학과 쌍)
15개 컬럼: No, 학과명, 학교명, 구분, 학교종류, 학교지역, 소재지, 소재지(상세), 설립구분, 주야구분, 학과특성, 표준분류계열(대), 표준분류계열(중), 표준분류계열(소), 관심

1. 새 테이블 생성
sqlCREATE TABLE university_majors (
  id INT AUTO_INCREMENT PRIMARY KEY,
  major_name VARCHAR(200) NOT NULL,       -- 학과명
  university_name VARCHAR(200) NOT NULL,  -- 학교명
  division VARCHAR(50),                    -- 구분 (대학, 전문대학 등)
  university_type VARCHAR(50),             -- 학교종류
  region VARCHAR(50),                      -- 학교지역
  location VARCHAR(100),                   -- 소재지
  location_detail VARCHAR(200),            -- 소재지(상세)
  establishment VARCHAR(50),               -- 설립구분 (국립, 사립 등)
  day_night VARCHAR(20),                   -- 주야구분
  major_characteristic TEXT,               -- 학과특성
  category_large VARCHAR(50),              -- 표준분류계열(대)
  category_medium VARCHAR(50),             -- 표준분류계열(중)
  category_small VARCHAR(100),             -- 표준분류계열(소)
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
2. 엑셀 데이터 임포트
학과 찾기.xlsx 파일을 읽어서 university_majors 테이블에 29,498개 행을 모두 INSERT 해라. 4년제 대학만 필터링하지 말고 전체를 넣어라 (전문대학 포함). 필터링은 조회 시점에 한다.
3. cached_majors와 연결
cached_majors의 전공명과 university_majors의 학과명을 매칭하여, 전공 상세 페이지에서 "이 전공을 제공하는 대학 목록"을 표시할 수 있도록 해라:

매칭 방식: cached_majors.name과 university_majors.major_name이 동일하거나 포함 관계인 경우
예: cached_majors에 "간호학"이 있으면, university_majors에서 학과명이 "간호학과", "간호학전공" 등을 매칭

4. universities 테이블과도 연결
university_majors.university_name과 기존 universities 테이블(489개)의 학교명을 매칭해라. 매칭되는 대학은 클릭 시 대학 상세 페이지로 이동할 수 있게 링크를 걸어라.
⚠️ 주의:

엑셀 데이터를 있는 그대로 임포트해라. 값을 변경하거나 보완하지 마라.
