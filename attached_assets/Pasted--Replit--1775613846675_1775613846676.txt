
Replit 지시사항: 진로 흥미 분석 시작 화면 레이아웃 개선

목표: 흥미 분석 시작 버튼의 클릭율을 높이고, 학과/직업 탐색 페이지와 시각적 통일감을 확보
현재 문제:

"흥미 분석 시작" 버튼이 페이지 최하단에 있어서 스크롤 없이 보이지 않을 수 있음
검사 구성 설명이 시작 버튼보다 먼저 큰 영역을 차지하여 행동 유도(CTA)가 약함
학과/직업 탐색 페이지의 간결한 배너 스타일과 달리, 진로 흥미 분석은 세로로 길게 나열됨

변경 방향: 학과/직업 탐색의 배너 스타일과 통일
학과/직업 탐색 페이지 상단에 "진로 흥미 분석" 배너가 있는 것처럼, 진로 흥미 분석 페이지도 동일한 구조로 변경해라:
[상단 영역 - 히어로 섹션] (스크롤 없이 바로 보이는 영역)

│                                                   │
│  커리어넷 443개 직업, 235개 학과 데이터를 기반으로    │
│  나에게 맞는 학과와 직업을 찾아보세요.               │
│                                                   │
│  약 5분 · 총 30문항                                │
│                                                   │
│  [흥미 분석 시작]  ← 큰 CTA 버튼                    │
│                                                   │
└─────────────────────────────────────────────────┘

[하단 영역 - 접힌 상태의 검사 구성 상세] (관심 있는 학생만 펼쳐 봄)
┌─────────────────────────────────────────────────┐

│                                                   │

핵심 변경:

"흥미 분석 시작" 버튼을 상단 히어로 영역 안에 배치 → 스크롤 없이 바로 보임
검사 구성 상세(9개 분야, 6개 역량 목록)는 접힌 상태로 기본 제공 → "검사 구성 자세히 보기" 클릭 시 펼침
커리어넷 안내 문구도 접힌 영역 안으로 이동
상단은 타이틀 + 한 줄 설명 + 소요시간 + CTA 버튼만 → 간결하고 행동 유도가 강함

역량 검사 약어 제거:
현재 역량 항목 앞에 VE, MA, SP, CR, SO, SE 같은 약어가 표시되고 있다. 이 약어들을 완전히 제거해라. 한글 역량명만 표시해라:

VE 언어능력 → 언어능력
MA 수리·논리력 → 수리·논리력
SP 공간·시각능력 → 공간·시각능력
CR 창의력 → 창의력
SO 대인관계능력 → 대인관계능력
SE 자기관리능력 → 자기관리능력

구현 체크리스트:

 상단 히어로 영역에 CTA 버튼 배치 (스크롤 없이 보임)
 검사 구성 상세를 접힌 상태로 변경
 역량 약어(VE, MA, SP, CR, SO, SE) 제거
 모바일에서도 CTA 버튼이 첫 화면에 보이는지 확인
